Ganga Bhumi Club - Tech Team Task
Event: Makar Sankranti Mela 2026 Registration System
This repository contains the solution for the Ganga Bhumi Club Tech Team technical task. The project is an integrated system for event registration, data management, and promotion for the "Makar Sankranti Mela 2026."

The project is divided into three main tasks as per the requirements.

Task 1: Google Form
A Google Form was created for event registration with the following features:

Title: "Ganga Bhumi Club - Makar Sankranti Mela 2026 Registration"

Validation: Includes specific email validation for @vitbhopal.ac.in and number validation for phone.

File Upload: Secure upload for payment screenshots (.jpg/.jpeg).

Required Fields: UTR Number, Name, Email, etc.

Task 2: Google Sheets Data Management
The response sheet linked to the Google Form was configured for professional data management:

New Column: A "Payment Status" column was added with dropdown values (Verified, Pending, Defaulter).

Conditional Formatting: The "UTR Number/UPI ID" column automatically highlights any blank cells to quickly identify missing payments.

Filters: Filters are enabled on the header row for easy sorting by Payment Status, Branch, etc.

Task 3: Landing Page
A clean, modern, and fully responsive landing page was built to promote the event.

Content: Features event branding, a "Register Now" button, and a scannable QR code.

Live Countdown: A JavaScript-powered timer actively counts down to the event start time (November 13, 2025, 09:00 AM).

Responsiveness: The design is optimized for desktop, tablet, and mobile devices.
